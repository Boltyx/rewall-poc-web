# Rewall POC - Current Scope

## Goal

Build a pitch-ready demo of Rewall - a social intelligence platform bridging academic research and community discussion.

## Active Focus

**Phase 1: Foundation Shell** - Setting up the visual skeleton and core design system.

## Key Decisions Made

- **Framework**: Vite + React
- **Styling**: Vanilla CSS with CSS Variables
- **Animations**: Framer Motion
- **Data**: Static JSON mocks + optional real APIs (Perplexity, Semantic Scholar)
- **Aesthetic**: Black/white minimalist, typography-driven, outlined buttons

## Pages to Build (Priority Order)

1. Landing Page (Discovery feed with search)
2. Search Results (AI synthesis view)
3. Topic Page (Spectrum visualization)
4. Source Reader (Paper with heatmap sidebar)
5. Publisher Page (Trust verification)

## Timeline

- Days, not weeks
- Live demo (controlled clicking)
- Foundation first, hero path refinement later
